# 按键事件



| 事件  |          说明，触发条件           |
| :---: | :-------------------------------: |
|  up   |      按键被按下后抬起，触发       |
| down  |      按键首次被按下后，触发       |
| short | 按键在被按下两百毫秒内可以，触发  |
| long  | 按键在被按下  两百 ~ 一秒内，触发 |
| even  |      按键被按下一秒后，触发       |

## 原型为：

```c
typedef enum {
    Up = 0u,
    Down,
    Short,
    Long,
    Even
}pressEvent;
```
